## 🎯 이 샘플의 특징

### 현실적인 설정
- 실제 학부 3학년생의 수준에 맞는 경험
- 구체적인 공정 파라미터 (압력, 온도, RF 파워 등)
- 실험 실패 → 개선 → 성공 과정 포함
- 현실적인 GPA와 활동 내역

### 반도체 공정 키워드 풍부
- **증착**: CVD, ALD, 스퍼터링, 열산화
- **식각**: RIE, 플라즈마, CF₄/O₂
- **리소그래피**: 포토레지스트, 노광
- **분석**: XRD, SEM, TEM, XPS, AFM
- **재료**: ITO, SiO₂, Al₂O₃
- **파라미터**: 압력, 온도, RF 파워, 가스 비율

### 맞춤형 질문 생성에 최적화
이력서/자소서를 분석하면 다음과 같은 맞춤형 질문이 생성됩니다:

**예상 질문 1:**
> "ITO 박막 프로젝트에서 RF 파워를 150W로 설정한 이유는 무엇인가요? 
> 더 높은 파워를 사용하면 어떤 문제가 발생할 수 있나요?"

**예상 질문 2:**
> "MEMS 센서 제작 시 RIE 식각에서 CF₄와 O₂의 혼합 비율을 어떻게 결정하셨나요? 
> 각 가스의 역할은 무엇인가요?"

**예상 질문 3:**
> "저온 ALD 공정에서 150℃를 선택한 이유와, 
> 고온 공정 대비 장단점을 설명하세요."

**예상 질문 4:**
> "XPS로 화학 조성을 분석한다고 했는데, 
> XPS의 원리와 어떤 정보를 얻을 수 있는지 설명하세요."

## 💻 시스템에서 테스트하기

### 1단계: 시뮬레이터 실행
```bash
python semiconductor_simulator.py
```

### 2단계: 프로필 설정
1. **"프로필 설정"** 탭 클릭
2. **이력서** 업로드: `sample_resume.docx`
3. **자기소개서** 업로드: `sample_personal_statement.docx`
4. **"분석 시작"** 클릭

### 3단계: 분석 결과 확인
시스템이 자동으로 추출하는 정보:

**학력:**
- 재료공학부 3학년
- GPA 3.82/4.3

**경험:**
- ITO 박막 증착 최적화 (스퍼터링)
- MEMS 센서 제작 (RIE, 리소그래피)
- 저온 ALD 공정 연구

**기술 스킬:**
- 증착: RF 스퍼터링, ALD, PECVD
- 식각: RIE, 습식 식각
- 분석: XRD, SEM, TEM, XPS

**관심 분야:**
- 박막 증착 공정
- 반도체 전공정
- 공정 최적화

**커리어 목표:**
- 대기업 공정 엔지니어
- 증착 공정 전문성

### 4단계: 맞춤형 면접
1. **"면접 모드"** 탭 클릭
2. **"내 프로필 기반 맞춤형 질문"** 체크
3. **"면접 시작"** 클릭

생성되는 질문 예시:
- "ITO 박막 프로젝트에서 어떤 어려움이 있었나요?"
- "RIE 식각 파라미터 최적화 과정을 설명하세요"
- "ALD의 장점을 CVD와 비교하여 설명하세요"

## 🔧 커스터마이징

### 자신의 이력서/자소서로 변경

1. **이력서 수정**
   - `generate_resume.py` 편집
   - 이름, 학교, GPA 변경
   - 프로젝트 경험 추가/수정
   - 실행: `python generate_resume.py`

2. **자기소개서 수정**
   - `generate_personal_statement.py` 편집
   - 동기, 경험, 목표 변경
   - 실행: `python generate_personal_statement.py`

### 샘플 수정 팁

**구체적인 수치 포함:**
```python
# 좋은 예
"RF 파워 150W, 압력 3mTorr, 온도 300℃에서 
 비저항 3.2×10⁻⁴ Ω·cm 달성"

# 나쁜 예
"적절한 조건에서 좋은 결과를 얻었음"
```

**실패 → 해결 과정 포함:**
```python
"처음에는 균일도가 불량했으나, 
 기판 회전 속도를 30rpm으로 높여 개선"
```

**구체적인 장비/기술 명시:**
```python
"박막 분석 장비: XRD (결정성), 
 4-point probe (전기전도도), 
 AFM (표면 거칠기)"
```
